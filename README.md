## ✨ Features

- **NFC Tag Interception:** Scans Plexamp NFC tags (HTTP/HTTPS URIs), preventing the system from opening broken links.
- **Universal Link Forging:** Automatically extracts the Server ID and Metadata ID from the tag and generates a working `playMedia` deep link.
- **Home Screen Widget (4x2):** A premium, glass-morphism widget (60% dark opacity, 20dp rounded corners) that displays a clean list of your saved playlists.
- **Playlist Management:** 
  - Drag & Drop to reorder playlists natively.
  - Easily rename or safely delete playlists directly from the app interface.
- **Zero Background Battery Drain:** NFC is strictly used during the initial configuration. The widget handles all daily playback triggers.

## 🛠 Tech Stack

- **Language:** Kotlin
- **Database:** Room (Local SQLite persistence)
- **UI:** XML Layouts & RemoteViews (for the AppWidget)
- **SDK:** Android `minSdk 26`, `targetSdk 34`
- **Build System:** Gradle (AGP 8.3.2)

## 🚀 How It Works

1. **Configuration:** Open the main app to manage your shortcuts.
2. **Add via NFC:** Click the Add button and scan an NFC tag previously written by Plexamp.
3. **URL Transformation:** The app reads the tag's URI, securely extracts the `source` and `key` parameters, and builds a corrected `playMedia` deep link with the `&shuffle=1` parameter added.
4. **Save & Customize:** Name your new shortcut. The modified URL, name, and sort order are saved locally in the Room database.
5. **Listen:** Add the widget to your home screen. Tap any playlist in the list to instantly wake up Plexamp and start the music.

## 🏗 Technical Decisions & UI Pivots

To keep the application lightweight, robust, and straight to the point, specific technical and design choices were made:

- **Strict URL Encoding:** Deep links are built using Android's native `Uri.Builder` to ensure special characters inside the nested URI are safely encoded before triggering the `ACTION_VIEW` intent.
- **Intent Execution:** Uses `FLAG_ACTIVITY_NEW_TASK` and `FLAG_ACTIVITY_CLEAR_TOP` to ensure Plexamp is properly brought to the foreground and receives the playback command even if it was already running in the background.
- **Minimalist Widget:** The widget strictly uses text. No icons are displayed on the widget to maintain a clean aesthetic, while the main configuration app includes standard management icons (edit/delete) for ease of use.
- **Safer Interactions:** The classic "Swipe-to-Delete" gesture was removed in favor of explicit Edit/Delete buttons and confirmation dialogs to prevent accidental deletions.

## 📥 Getting Started

### Prerequisites
- Android Studio (Iguana or newer recommended)
- JDK 17+ (Verified compatibility with JDK 21)
- Android SDK 34

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/PlexampPlaylistWidget.git
   ```
2. Open the project in Android Studio.
3. Sync the Gradle files.
4. Build and run the app on your Android device.
